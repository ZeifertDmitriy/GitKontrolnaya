## README

### Программа, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символам.

- Для выполнения программы введите по очереди четыре строки с разным количеством символов.
- В результате Будет выдан новый массив с количеством строк равным количеству строк  первого массива с колличеством символов равны либо меньшим трем.
